# Cognik Intent Interpreter

## Função

Transformar intenção curta do usuário em escopo claro de projeto.

## Input mínimo

> Criar projeto para perfil no Instagram de uma recém-advogada penal feminina no Brasil.

## O que Cognik deve inferir

- Projeto de personal branding.
- Nicho regulado e sensível.
- Risco ético alto.
- Necessidade de pesquisa normativa.
- Necessidade de linguagem segura.
- Possível output: análise, plano editorial, HTML, prompts visuais, checklist ético.

## Perguntas que Cognik deve fazer antes de execução final

1. Miriam já tem OAB ativa?
2. Qual cidade/estado?
3. Ela quer parecer mais técnica, acolhedora, firme ou institucional?
4. Ela já tem Instagram?
5. Há restrições de exposição pessoal?
6. Quais temas ela não quer abordar?
7. O objetivo inicial é autoridade, clientes, networking, estágio, sociedade ou conteúdo educativo?

## Output esperado

Um briefing estruturado com:

- objetivo;
- riscos;
- escopo;
- hipóteses;
- lacunas;
- agentes recomendados;
- custo simulado;
- próximo passo para aprovação.
