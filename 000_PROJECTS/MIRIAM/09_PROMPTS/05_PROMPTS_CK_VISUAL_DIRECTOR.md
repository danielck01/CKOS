# Prompts para CK Visual Director

Estes prompts não geram imagem diretamente. Eles pedem ao Diretor Visual criar engenharia de prompt completa.

## 1. Miriam editorial jurídico sóbrio

```md
Atue como CK Visual Director. Crie uma engenharia de prompt completa para uma imagem editorial premium de uma jovem advogada penal brasileira em ambiente sóbrio, escuro, elegante, com presença firme e humana. Evitar clichês jurídicos como martelo, algemas, sangue, cela, tribunal dramático ou estética policial. A imagem deve transmitir clareza, ética, autoridade jovem e confiança. Incluir direção de câmera, lente, iluminação, composição, materialidade, paleta, prompt negativo e critérios de aprovação.
```

## 2. Ética e autoridade

```md
Atue como CK Visual Director. Crie uma engenharia de prompt para representar visualmente o equilíbrio entre ética jurídica e autoridade profissional. Fundo preto, glassmorphism sutil, elementos abstratos de luz, documentos e linhas de decisão. Sem símbolos jurídicos óbvios. A sensação deve ser de governança, sobriedade, responsabilidade e confiança.
```

## 3. Matriz de risco reputacional

```md
Atue como CK Visual Director. Crie uma engenharia de prompt para uma imagem conceitual que represente uma matriz de risco reputacional para advocacia penal: baixo risco, médio risco, alto risco e bloqueado. Estética dark editorial, visual limpo, camadas translúcidas, sem texto legível na imagem. Deve servir como background para cards glass no HTML.
```

## 4. Presença feminina no direito penal

```md
Atue como CK Visual Director. Crie uma engenharia de prompt para uma imagem simbólica de presença feminina firme no direito penal, sem caricatura de heroína, sem agressividade e sem sexualização. A estética deve ser madura, silenciosa, forte, humana e técnica. Fundo escuro, luz lateral, postura segura.
```

## 5. Linha editorial como cockpit

```md
Atue como CK Visual Director. Crie uma engenharia de prompt para visualizar uma linha editorial jurídica como um cockpit cognitivo: cards, pilares, risco ético, retenção, clareza e confiança. Sem UI real legível, apenas sensação abstrata de sistema estratégico. Fundo preto, vidro, luzes discretas, premium.
```
