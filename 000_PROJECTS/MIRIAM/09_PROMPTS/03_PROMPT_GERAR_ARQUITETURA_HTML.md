# Prompt para Arquitetura do HTML

```md
Com base nas 3 Deep Research e no Miriam Material Build Pack, gere apenas a arquitetura do HTML, sem escrever o código final ainda.

Objetivo:
Criar uma página de estudo/apresentação sobre Miriam, personal branding jurídico para advogada penal recém-formada.

A arquitetura deve conter:
- ordem das seções;
- função de cada seção;
- componentes visuais;
- tipos de diagramas;
- exercícios;
- prompts visuais copiáveis;
- pontos de ética;
- matriz de risco;
- plano de ação;
- critérios de QA.

Restrições:
- bg black;
- mobile-first;
- linguagem fácil;
- sem imagem real;
- wireframes para imagens futuras;
- botão copiar prompt nos blocos de imagem;
- sem backend;
- sem React;
- sem promessas jurídicas.

Antes de finalizar, marque:
- fatos das fontes;
- hipóteses estratégicas;
- pontos a validar com Miriam.
```
