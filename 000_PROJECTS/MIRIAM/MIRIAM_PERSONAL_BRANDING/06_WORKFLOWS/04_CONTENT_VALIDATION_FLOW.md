# Content Validation Flow

Status: future_workflow

## Funcao

Validar qualquer conteudo antes de virar pauta, post, roteiro ou legenda.

## Fluxo

```txt
content_idea
-> risk_classifier
-> claim_validation
-> OAB language check
-> reputational check
-> human review when needed
-> approved_draft_or_blocked
```

## Checks minimos

- informativo vs captacao;
- clareza leiga;
- ausencia de promessa;
- ausencia de sensacionalismo;
- cuidado com caso concreto;
- adequacao a experiencia real de Miriam.

## Status

Nao executar enquanto estrategia base nao estiver aprovada.
