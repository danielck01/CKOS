# UI/UX Reference Notes

Status: reference_only

## Leitura das referencias visuais

As imagens anexadas apontam para uma direcao de produto mais interativa que um documento estatico:

- commandbar como ponto de entrada da intencao;
- cards glassmorphism;
- widgets mobile com informacao curta;
- status chips e microinteracoes;
- cards flutuantes de decisao;
- telas em formato de celular;
- uso de fundo escuro com acentos vermelho, verde, rosa e amarelo acido;
- narrativa visual com telefone, notas, alertas e objetos simbolicos;
- interface que parece uma sala de decisao, nao uma landing page de marketing.

## Traducao para CKOS

| Referencia | Traducao no prototipo |
| --- | --- |
| AI components / commandbar | Campo de intencao curta e simulacao CEO Agent |
| Mobile widgets | Cards de status, gate, output e ROI |
| Phone screen flows | Scroll mobile em secoes curtas |
| Red phones | Simbolo de chamadas, decisoes e perguntas pendentes |
| Sticky notes | Gaps, tasks e pontos a validar |
| Glass cards | Artifacts e QA cards |

## Regra de uso

As imagens sao referencia de atmosfera e UX. Nao copiar marcas, layouts proprietarios ou textos. O CDN futuro deve usar imagens autorizadas.
