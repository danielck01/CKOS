# WhatsApp Send Note

Status: operational_note

## Como enviar

1. Compactar a pasta `10_HTML_MOBILE_PACK`.
2. Enviar o arquivo `.zip` pelo WhatsApp.
3. No celular, extrair o zip.
4. Abrir `index.html` no navegador.

## Observacoes

- O arquivo funciona offline.
- Imagens CDN so aparecerao depois de preencher `assets/cdn-image-map.js`.
- Se o WhatsApp bloquear abertura direta do HTML, enviar tambem por Google Drive ou abrir depois de extrair no gerenciador de arquivos.

## Arquivo compactado sugerido

`MIRIAM_HTML_MOBILE_PACK_WHATSAPP.zip`

## O que nao acontece automaticamente

- Codex nao envia pelo WhatsApp.
- O HTML nao sobe arquivos para a internet.
- O HTML nao grava respostas em Google Drive.
- O Google Drive e opcional e precisa do script separado.
