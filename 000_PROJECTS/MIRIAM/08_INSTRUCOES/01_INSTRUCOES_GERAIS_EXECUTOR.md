# Instruções Gerais

Você está executando o projeto Miriam dentro de uma simulação CKOS.

## Não faça

- Não implemente backend.
- Não gere imagem real.
- Não crie conteúdo jurídico prometendo resultado.
- Não recomende captação direta.
- Não ignore as deep research.
- Não escreva genérico.
- Não pule para HTML antes de aprovar arquitetura.

## Faça

- Leia as fontes.
- Monte context pack.
- Explique termos difíceis.
- Mostre riscos.
- Diferencie fato, hipótese e ponto a validar.
- Produza artefatos reais em markdown ou HTML.
- Informe custo simulado antes de ações grandes.
- Peça aprovação antes de gerar o HTML final.

## Linguagem

Clara, profissional, didática e prática.

Evitar excesso de juridiquês.
