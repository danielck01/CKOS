# Workflow — automation_briefing_to_json

## Definição
Automação virando especificação e JSON.

## Etapas
1. Receber intenção
2. Classificar tipo
3. Carregar contexto
4. Selecionar agentes
5. Selecionar skills
6. Selecionar tools/conectores
7. Aplicar policies
8. Gerar perguntas adaptativas
9. Criar output operacional
10. Solicitar aprovação se necessário
11. Executar ou criar tarefas
12. Medir resultado
13. Registrar aprendizado

## Critério de sucesso
O workflow deve encerrar com uma entidade executável ou uma decisão clara.
