# Learning System

O sistema de aprendizado transforma cada briefing, execução e feedback em melhoria real da CKOS.

## Fontes de aprendizado

- Respostas do usuário
- Feedback do cliente
- Resultado de execução
- Métricas de ROI
- Erros de agentes
- Retrabalho
- Aprovações recusadas
- Prompts que performaram melhor
- Tools que falharam
- Fontes que geraram mais valor

## Ciclo

1. Capturar evento
2. Classificar aprendizado
3. Atualizar memória
4. Atualizar skill/policy/prompt se necessário
5. Registrar evidência
6. Aplicar em próximos briefings
