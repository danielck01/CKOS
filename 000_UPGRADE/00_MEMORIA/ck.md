# ck.md — Memória Operacional da Pasta

## Nome
CKOS Universal AI FIRST Briefing Runtime

## Definição
Sistema universal de briefing da CKOS que transforma qualquer intenção em execução estruturada, mensurável e aprendível.

## Correção estratégica
Este sistema não é um briefing apenas para criação de projetos.
Ele é um runtime universal aplicável a qualquer tarefa, projeto, campanha, conteúdo, pesquisa, automação, proposta ou decisão.

## Tese
Da intenção ao ROI.

## Sequência universal
Intenção → Classificação → Contexto → Research → Agentes → Skills → Tools → Policies → Briefing Adaptativo → Execução → Artefatos → Aprovação → ROI → Aprendizado.

## Diferença
Um formulário coleta dados.
Um chatbot responde.
Um RAG recupera contexto.
O CKOS Briefing Runtime coordena inteligência, contexto, ferramentas, agentes, decisões e aprendizado.

## Princípios obrigatórios
- Não executar sem classificar intenção.
- Não perguntar sem saber por que pergunta.
- Não usar apenas LLM quando ferramentas, fontes ou dados forem necessários.
- Não gerar proposta sem diagnóstico.
- Não gerar tarefa sem critério de sucesso.
- Não concluir briefing sem output acionável.
- Não encerrar execução sem feedback e aprendizado.
