# Agent Civilization

A civilização da CKOS organiza agentes por nível de responsabilidade.

## Níveis

- Superagents: comandam áreas inteiras.
- Agents: executam capacidades especializadas.
- Subagents: realizam microtarefas.
- Reviewers: validam qualidade, risco e aderência a policies.
- Human Gates: aprovam decisões críticas.

## Regra
Nenhum briefing complexo deve ser respondido por um único agente generalista.
