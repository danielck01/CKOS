# Subagent — quality_checker

## Função
Executar uma microtarefa especializada dentro do briefing.

## Entrada
- Context State
- Trecho analisado
- Objetivo do agente principal

## Saída
- Resultado curto
- Status
- Confiança
- Sugestão de próxima ação

## Regra
Subagents não definem estratégia global. Eles alimentam agents e superagents.
