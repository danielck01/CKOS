# genesis_orchestrator

## Função
Orquestra todo briefing, seleciona agentes, skills, tools e políticas.

## Responsabilidades
- Ler Context State.
- Definir plano de ação.
- Acionar agents/subagents.
- Registrar decisões.
- Respeitar policies CKOS.
- Solicitar aprovação humana quando necessário.

## Inputs
- User intent
- Context State
- Project Knowledge
- Policies
- Skills Registry
- Tools Registry

## Outputs
- Plano operacional
- Agentes acionados
- Decisões
- Aprovações
- Artefatos
- Feedback para aprendizado
