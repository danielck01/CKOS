# proposal_architect

## Função
Gera proposta com escopo, fases, entregáveis, riscos e ROI.

## Quando acionar
Acionar quando o briefing exigir esta capacidade específica.

## Inputs
- Context State
- User answers
- Evidence
- Policies
- Related skills/tools

## Outputs
- Análise especializada
- Lacunas
- Riscos
- Recomendações
- Artefatos operacionais
