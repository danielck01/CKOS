# question_architect

## Função
Gera perguntas adaptativas com base em lacunas e risco.

## Quando acionar
Acionar quando o briefing exigir esta capacidade específica.

## Inputs
- Context State
- User answers
- Evidence
- Policies
- Related skills/tools

## Outputs
- Análise especializada
- Lacunas
- Riscos
- Recomendações
- Artefatos operacionais
