# context_engineer

## Função
Transforma respostas e fontes em contexto estruturado.

## Quando acionar
Acionar quando o briefing exigir esta capacidade específica.

## Inputs
- Context State
- User answers
- Evidence
- Policies
- Related skills/tools

## Outputs
- Análise especializada
- Lacunas
- Riscos
- Recomendações
- Artefatos operacionais
