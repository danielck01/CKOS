# campaign_strategist

## Função
Monta campanhas com narrativa, canais, criativos, orçamento e mensuração.

## Quando acionar
Acionar quando o briefing exigir esta capacidade específica.

## Inputs
- Context State
- User answers
- Evidence
- Policies
- Related skills/tools

## Outputs
- Análise especializada
- Lacunas
- Riscos
- Recomendações
- Artefatos operacionais
