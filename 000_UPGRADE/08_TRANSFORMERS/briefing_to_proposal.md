# Transformer — briefing_to_proposal

## Função
Transforma briefing em proposta.

## Input
- Context State
- Dados brutos
- Tipo de briefing
- Policies aplicáveis

## Output
- Entidade transformada
- Confiança
- Lacunas
- Próximas ações

## Exemplo de uso
O sistema recebe uma resposta ambígua, estrutura em YAML, detecta lacunas e gera pergunta seguinte ou output operacional.
