# Transformer — answer_to_context

## Função
Transforma resposta do usuário em contexto estruturado.

## Input
- Context State
- Dados brutos
- Tipo de briefing
- Policies aplicáveis

## Output
- Entidade transformada
- Confiança
- Lacunas
- Próximas ações

## Exemplo de uso
O sistema recebe uma resposta ambígua, estrutura em YAML, detecta lacunas e gera pergunta seguinte ou output operacional.
