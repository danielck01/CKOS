# CKOS Infra Pack — DigitalOcean + n8n Automations

Criado em: 2026-05-26

Objetivo:
Estruturar notas operacionais para o Codex/Claude Code usar como contexto na migração online do ecossistema CKOS, priorizando nascimento enxuto, uso inteligente de crédito inicial, VPS performática, n8n como acelerador e migração gradual para código próprio.

Princípio central:
O projeto precisa nascer pequeno, provar valor, capturar receita e se pagar sozinho. Crédito grátis não é dinheiro para gastar. É runway estratégico para validar infraestrutura, automações, aquisição, proposta, ROI e execução.

Uso recomendado:
1. Ler `04_Codex_Instructions/00_Contexto_Mestre_para_Codex.md`.
2. Ler a nota de decisão sobre DigitalOcean.
3. Usar as notas de n8n como prompts de geração de automações.
4. Importar/ajustar os JSONs no n8n somente após revisar credenciais e webhooks.
5. Migrar automações críticas para código próprio quando atingirem recorrência, risco ou impacto financeiro.
