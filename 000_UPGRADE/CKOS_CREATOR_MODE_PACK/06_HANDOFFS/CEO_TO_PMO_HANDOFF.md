# CEO_TO_PMO_HANDOFF

## Uso

Quando o CEO Agent precisar de auditoria antes de propor execucao ao Founder.

## Template

```txt
PMO, revise este plano antes de eu propor execucao ao Founder.

Project:
Intent:
Project type:
Risk:
Estimated CKOS credits:
Source mode:
Proposed output:
Allowed files:
Forbidden files:
Known risks:
Open questions:
Decision requested:
```

## Expectativa

PMO retorna:

- aprovado;
- aprovado com patches;
- bloqueado.
