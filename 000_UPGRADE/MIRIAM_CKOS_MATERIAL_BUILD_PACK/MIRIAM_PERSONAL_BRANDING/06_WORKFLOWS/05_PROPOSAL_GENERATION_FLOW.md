# Proposal Generation Flow

Status: future_workflow

## Funcao

Definir quando o projeto pode virar proposta, guia ou plano de acao para Miriam.

## Pre-condicoes

- briefing vivo com respostas reais;
- source evidence map;
- matriz de riscos eticos;
- decisao de perfil;
- limites de linguagem;
- PMO audit;
- aprovacao Founder/Miriam.

## Saidas futuras possiveis

- analise estrategica;
- guia de presenca;
- plano de acao preliminar;
- proposta de proximas etapas.

## Bloqueio

Nao gerar proposta final nesta fase.
