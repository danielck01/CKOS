# Créditos Simulados

Esta nota simula como o CKOS poderia avisar custo antes de executar.

## Tabela inicial

| Ação | Créditos simulados | Observação |
|---|---:|---|
| Ler fontes e montar context pack | 8 | obrigatório |
| Diagnóstico estratégico | 12 | inclui riscos e tese |
| Linha editorial | 10 | inclui exemplos |
| Arquitetura HTML | 8 | sem gerar HTML ainda |
| Gerar HTML completo | 22 | alto contexto |
| QA Metacognik | 8 | obrigatório por risco jurídico |
| Prompts visuais | 10 | sem gerar imagens |
| Pack final de notas | 12 | opcional |

## Regra

O CEO Agent deve mostrar custo antes da execução.

Exemplo:

> Esta ação consumirá aproximadamente 42 créditos simulados. Deseja aprovar a execução ou gerar apenas o plano primeiro?

## Modo econômico

Para economizar créditos:

1. gerar apenas plano;
2. aprovar filetree;
3. gerar notas por lote;
4. gerar HTML por último.
