# Critérios de Aprovação

## Aprovar se

- Explica o problema com clareza.
- Usa as deep research como base.
- Tem exemplos práticos.
- Tem plano de ação.
- Tem exercícios.
- Tem matriz de risco.
- Tem prompts visuais copiáveis.
- É legível no celular.
- Respeita OAB e ética jurídica.
- Não promete resultado.

## Revisar se

- Ficar bonito, mas raso.
- Tiver jargão demais.
- Faltar exemplos.
- Faltar distinção entre hipótese e fato.
- O visual parecer advocacia clichê.
- O conteúdo parecer captação.

## Bloquear se

- Prometer resultado.
- Explorar medo.
- Usar crime como espetáculo.
- Expor caso concreto.
- Sugerir consulta personalizada gratuita.
- Usar linguagem de garantia.
