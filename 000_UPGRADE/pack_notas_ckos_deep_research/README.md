# Pack de Notas — Deep Research CKOS

Este pack contém notas em Markdown com engenharia de prompt para aprofundar o CKOS como sistema operacional AI-first da CKCompany.

## Contexto operacional

- Estamos codando localmente.
- O ecossistema CKOS será migrado para uma VPS.
- Já existem CDN e host.
- As notas foram pensadas para alimentar Claude Code, VibeCoding, Base44 ou outro agente de desenvolvimento/produto.
- O foco é execução real: arquitetura, produto, UX, agentes, propostas, pagamentos, suporte, governança e infraestrutura.

## Como usar

1. Abra uma nota por vez.
2. Cole o bloco `Prompt principal` no Claude Code ou na ferramenta de IA.
3. Após a resposta, peça:  
   `Transforme essa pesquisa em tarefas técnicas, modelo de dados e componentes React.`
4. Depois peça:  
   `Agora gere o plano de implementação em fases para ambiente local, staging e VPS.`
5. Salve a resposta como nova nota no vault do CKOS.

## Regra central

O CKOS não deve parecer um app padrão. Ele deve parecer um control plane cognitivo/metacognitivo que transforma intenção em projeto, proposta, execução, agentes, decisões, orçamento, ROI e aprendizado contínuo.
