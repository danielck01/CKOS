# Output Engine

O Output Engine decide o formato final do briefing.

## Outputs possíveis

- Markdown pack
- Proposta comercial
- Task card
- Sprint
- Roteiro
- Storyboard
- Dashboard
- Documento estratégico
- Prompt para Claude Code
- JSON de automação
- Issue técnica
- Plano de pesquisa
- Briefing criativo
- Brand artifact
- Checklist de produção
- Decision memo

## Regra
Todo briefing deve terminar com pelo menos um output acionável.
