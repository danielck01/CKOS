# Visão Geral — Universal Briefing Runtime

O Universal Briefing Runtime é o motor que recebe qualquer intenção e decide como transformá-la em execução.

## Entrada
Mensagem, áudio, arquivo, imagem, link, documento, print, vídeo, comando ou seleção em interface.

## Saída
Pode gerar:

- projeto;
- tarefa;
- subtarefa;
- proposta;
- sprint;
- roteiro;
- plano;
- pesquisa;
- dashboard;
- nota .md;
- workflow;
- campanha;
- briefing de produção;
- issue técnica;
- artefato visual;
- automação;
- decisão;
- aprovação pendente;
- aprendizado.

## Pipeline universal

1. Capture Intent
2. Classify Execution Type
3. Load Context
4. Detect Missing Context
5. Trigger Research/Tools
6. Select Agents
7. Select Skills
8. Apply Policies
9. Generate Adaptive Briefing
10. Transform Answers
11. Create Operational Output
12. Request Approval if needed
13. Execute or Schedule
14. Measure Outcome
15. Learn
