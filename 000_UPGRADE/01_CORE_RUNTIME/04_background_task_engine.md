# Background Task Engine

O Briefing AI FIRST precisa executar tarefas simultâneas enquanto conversa.

## Exemplos

Enquanto o usuário responde:
- Research Agent busca referências.
- Apify Operator coleta dados.
- Context Engineer atualiza memória.
- Risk Analyst detecta contradições.
- Prompt Engineer normaliza resposta.
- Proposal Agent prepara esqueleto.
- ROI Analyst estima impacto.
- QA Agent valida qualidade.

## Status visíveis

- analisando intenção;
- buscando fontes;
- comparando referências;
- estruturando contexto;
- detectando lacunas;
- calculando ROI;
- preparando proposta;
- aguardando aprovação.

## Regra de UX
Mostrar progresso vivo, mas sem poluir a tela. A timeline deve parecer Manus/agent journey, não log técnico bruto.
