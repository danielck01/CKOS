# AI First Project Planner Skill

## Função

Criar documentação viva de projeto com mínima intervenção humana.

## O que cada projeto deve gerar automaticamente

- Project brief.
- Project architecture.
- Stakeholder map.
- Risks and gaps.
- Decision log.
- Agent squad proposal.
- Skill map.
- Workflow map.
- Cost estimate.
- ROI hypothesis.
- Support plan.
- Learning mode summary.
- Roadmap.
- Sprint plan.
- Artifact index.

## Modos do CKOS

### Planning Mode

Baixo custo, sem execução. Serve para entender, estruturar e simular.

### Smart Mode

Cognik monta contexto, Metacognik audita riscos, agentes sugerem caminhos.

### Execution Mode

Executa workflows aprovados, consome créditos e gera outputs.

### Audit Mode

Revisa decisões, custo, riscos, evidências e qualidade.

## Regra

O usuário não deve precisar enviar prompts longos para Claude. O CKOS deve criar e manter o contexto, pedir aprovação e orientar o Founder.
