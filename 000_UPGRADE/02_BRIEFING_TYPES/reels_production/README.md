# Briefing Type — reels_production

## Definição
Briefing para produção de Reels/TikTok com objetivo, gancho, narrativa, cena, roteiro, asset, publicação e métrica.

## Estrutura mínima
- Intenção
- Tipo de execução
- Contexto necessário
- Perguntas adaptativas
- Agentes envolvidos
- Skills necessárias
- Tools/conectores acionáveis
- Policies aplicadas
- Output esperado
- Critérios de sucesso
- Métrica de ROI
- Feedback/aprendizado

## Saída operacional
Este briefing deve gerar uma entidade executável: projeto, tarefa, sprint, roteiro, campanha, proposta, issue, documento, automação ou artefato.
