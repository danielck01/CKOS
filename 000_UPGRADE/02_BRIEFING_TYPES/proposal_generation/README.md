# Briefing Type — proposal_generation

## Definição
Geração de proposta a partir de diagnóstico, escopo, valor, prazo, riscos, entregáveis e plano de execução.

## Estrutura mínima
- Intenção
- Tipo de execução
- Contexto necessário
- Perguntas adaptativas
- Agentes envolvidos
- Skills necessárias
- Tools/conectores acionáveis
- Policies aplicadas
- Output esperado
- Critérios de sucesso
- Métrica de ROI
- Feedback/aprendizado

## Saída operacional
Este briefing deve gerar uma entidade executável: projeto, tarefa, sprint, roteiro, campanha, proposta, issue, documento, automação ou artefato.
