# cost_policy

## Definição
Controla custo por modelo, agente, ferramenta, run e projeto.

## Regras base
- Aplicar antes de execução.
- Registrar decisão.
- Sinalizar risco.
- Solicitar aprovação se necessário.
- Alimentar aprendizado após conclusão.

## Checklist
- Há dados suficientes?
- Existe risco?
- Existe custo?
- Precisa de fonte?
- Precisa de aprovação?
- Qual métrica de sucesso?
